## 💻 실행 방법

1. **원격 저장소 복제**

```bash
$ git clone https://github.com/JeongHwan-dev/movie-app-with-vue
```

2. **프로젝트 폴더로 이동**

```bash
$ cd movie-app-with-vue
```

3. **필요한 node_modules 설치**

```bash
$ npm install
```

4. **개발 서버 실행**

```bash
$ npm run dev
```
